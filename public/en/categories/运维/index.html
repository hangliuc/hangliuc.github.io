MySQL is a popular relational database management system (RDBMS) that is widely used for various applications. Here are some commonly used MySQL commands: ### Database Management - **Create Database**: `CREATE DATABASE database_name;` - **Show Databases**: `SHOW DATABASES;` - **Use Database**: `USE database_name;` - **Drop Database**: `DROP DATABASE database_name;` ### Table Management - **Create Table**: `CREATE TABLE table_name (column1 datatype, column2 datatype, ...);` - **Show Tables**: `SHOW TABLES;` - **Drop Table**: `DROP TABLE table_name;` - **Describe Table**: `DESCRIBE table_name;` ### Data Manipulation - **Insert Data**: `INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);` - **Select Data**: `SELECT column1, column2, ... FROM table_name;` - **Update Data**: `UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;` - **Delete Data**: `DELETE FROM table_name WHERE condition;` ### Data Retrieval - **Select with Conditions**: `SELECT * FROM table_name WHERE condition;` - **Ordering Data**: `SELECT * FROM table_name ORDER BY column1 ASC|DESC;` - **Limiting Results**: `SELECT * FROM table_name LIMIT offset, row_count;` ### User Management - **Create User**: `CREATE USER 'username'@'host' IDENTIFIED BY 'password';` - **Grant Privileges**: `GRANT privileges ON database_name.* TO 'username'@'host';` - **Revoke Privileges**: `REVOKE privileges ON database_name.* FROM 'username'@'host';` - **Show Users**: `SHOW USER;` ### Backup and Restore - **Backup Database**: `mysqldump -u username -p database_name > backup_file.sql;` - **Restore Database**: `mysql -u username -p database_name < backup_file.sql;` These commands provide a foundation for managing databases, tables, and data within MySQL.

Vim is a powerful text editor that is widely used in the Linux community. Here are some Vim tips and tricks to help you become more efficient: ## Basic Navigation - `h`: Move the cursor left - `j`: Move the cursor down - `k`: Move the cursor up - `l`: Move the cursor right - `0`: Move the cursor to the beginning of the line - `$`: Move the cursor to the end of the line - `gg`: Move the cursor to the beginning of the file - `G`: Move the cursor to the end of the file ## Text Editing - `i`: Enter insert mode - `a`: Enter append mode - `o`: Open a new line below and enter insert mode - `O`: Open a new line above and enter insert mode - `cw`: Change the word under the cursor - `cc`: Change the entire line - `r`: Replace the character under the cursor - `x`: Delete the character under the cursor - `dd`: Delete the entire line ## Multiple Selection - `Ctrl+v`: Enter visual mode - `i`: Enter insert mode within visual mode - `a`: Enter append mode within visual mode - `o`: Open a new line below and enter insert mode within visual mode - `O`: Open a new line above and enter insert mode within visual mode - `cw`: Change the selected text - `cc`: Change the entire selected lines - `x`: Delete the selected text ## Search and Replace - `/{pattern}`: Search for a pattern in the file - `n`: Go to the next match - `N`: Go to the previous match - `:%s/old/new/g`: Replace all occurrences of "old" with "new" in the file - `:s/old/new/g`: Replace all occurrences of "old" with "new" in the current line ## Marks - `m{letter}`: Set a mark at the current position - `'{letter}`: Go to the mark set with the same letter - ``{letter}`: Go to the mark set with the same letter and stay there ## Window Management - `:split`: Split the window horizontally - `:vsplit`: Split the window vertically - `Ctrl+w w`: Switch between windows - `Ctrl+w q`: Close the current window ## Command Line - `:help`: Display help for a command - `:version`: Display the version of Vim - `:q`: Quit Vim - `:wq`: Write and quit Vim These are just a few Vim tips and tricks to get you started. There are many more features and commands to explore. Happy editing!
